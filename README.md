# lab-bet365-lab

本仓库用于归档和发布多个独立的 HTML 页面，作为前端页面素材的集中存放地点。

## 项目简介

`lab-bet365-lab` 是一个轻量级的前端页面归档仓库，主要存放各类独立开发的 HTML 文件。这些页面不针对任何特定网站或服务，仅作为页面模板、示例或实验性内容的存档与展示。

## 目录说明

```
.
├── pages/          # 存放所有 HTML 页面文件
├── assets/         # 页面引用的公共资源（CSS、JS、图片等）
├── docs/           # 文档或说明文件（可选）
└── README.md       # 本文件
```

- `pages/` 目录下每个 HTML 文件均为独立页面，互不依赖。
- `assets/` 目录存放页面可能用到的通用资源文件，按需引用。

## 页面归档说明

- 所有页面均为静态 HTML，无后端依赖。
- 页面之间无关联，可单独打开使用。
- 归档内容可能不定期更新或清理，以保持仓库整洁。

## 维护说明

- 欢迎提交 Issue 或 Pull Request 来补充或修正页面内容。
- 建议新增页面时遵循以下规范：
  - 页面文件放置于 `pages/` 目录下。
  - 文件名使用英文小写，单词间以短横线连接（如 `example-page.html`）。
  - 页面内引用资源时优先使用相对路径。
- 本仓库不涉及任何商业用途，仅作为技术存档与展示。

## 注意事项

- 页面内容仅供学习与参考，不构成任何形式的建议或承诺。
- 请勿将本仓库内容用于违反相关法律法规的用途。
